# Tamil-Nadu-Legislative-Election-Dashboard-1971---2021


##  Project Overview

This project presents a **Tamil Nadu Legislative Assembly Election Dashboard** analyzing election data from **1971 to 2021**.
The dashboard visualizes voter statistics, party performance, constituency voting patterns, and winning candidate details.

The goal of this project is to provide **clear insights into Tamil Nadu election trends using data visualization.**

---



---

##  Key Metrics

* **Total Voters:** 60 Million
* **Total Votes Cast:** 46 Million
* **Total Constituencies:** 234

---

##  Dashboard Insights

### Total Votes by Constituency

Shows vote distribution across different constituencies in Tamil Nadu.

### Total Votes by Winning Candidates

Displays candidates with the highest number of votes.

### Total Votes by Party

Compares vote share among major political parties.

### Winning Percentage by Party

Shows which political parties have the highest winning percentages.

### Total Votes and Margin by Party

Analyzes the vote margin trends between major political parties.
